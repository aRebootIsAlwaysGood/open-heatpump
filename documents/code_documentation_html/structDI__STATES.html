Ausführliche Beschreibung

Definition Strukturtyps zum Speichern von 8 binären Werten in Bitfeldern.

Dieser Strukturtyp kann als Variable zum Ablegen von Digitaleingangszuständen genutzt werden. Es können Statusinfos für acht digitale Steuereingänge abgelegt werden. Diese Strukturtyp Definition wurde für eine spezifische Wärmepumpe geschrieben und enthält deshalb spezifische Membernamen.
