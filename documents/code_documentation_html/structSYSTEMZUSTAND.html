Ausführliche Beschreibung

Definiert einen 16-Bitfield Variablentyp für Informationen.

Innerhalb solchen Strukturen können Variablen mit nur zwei Werten speichersparend abgelegt werden, beispielsweise zum Systemzustand. Der Aufruf eines Members kann wie bei Klassenfunktionen erfolgen.
