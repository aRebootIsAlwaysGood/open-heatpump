◆ blinkFunction()

void blinkFunction ( )

Blinkfunktion, welche ein binäres Taktsignal von 1Hz liefert.

Globale Blinkvariable gesteuert durch blinkFunction().
Mittels der Blinkfunktion wird ein ständiges Taktsignal von 1Hz erzeugt, welches die globale Variable blink1Hz aktualisiert. Somit kann diese eingelesen werden und somit entfällt die ständige Notwendigkeit, auf die millis() Funktion zurückzugreifen.

Definiert in Zeile 104 der Datei wpMain.ino.
