Dokumentation der Funktionen

◆ speicherladung()

void speicherladung ( float  t_vorlaufsoll)

Führt die Speicherladung und Abtauung mittels Zustandsautomat durch.

Kontrolliert die Speichertemperatur und hält sie stets oberhalb der Vorlauftemperatur durch initiieren eines Ladevorgangs. Die Funktion steuert ausserdem den Abtauvorgang und leitet ihn in Abhängigkeit von Laufzeit und Aussentemperatur selbständig ein. Die Richtwerte entstammen aus Fachliteratur.
Untenstehend sind die Überlegungen in Notizform niedergeschrieben:
Stufenladung Speicher monovalent RAVEL Empfehlung: Siehe RAVEL Standardschaltungen S16 Temperaturdiff über Verflüssiger = 0.5...0.7*Tempdiff VL/RL im Auslegepunkt(45-35=10°C) = 5°C -> Ausschalten WP wenn Tkondensatorrücklauf - Tspeicher < 5...7°C –>witterungsgeführtes Ausschalten: (Tspeicher - Tkond) < 5°C && Tkond < 46°C Einstellregeln:

  1. Speichertemp bei WP-Start(Einschalttemp) <= Speichertemp bei WP-Stop(Ausschalttemp)
    1. Ausschalttemp <= max. zul. Verflüssiger-Austrittstemp - max Tempdiff über Verfl. -> 50*C - (5...7°C) = 45°C (max Heizanforderung)
    2. Bedingung: Einschalttemp >= max. auftretende Rücklauftemp -> Ton >= (Tvorlauf-Tempdiff_Heizung) falls > Tspeicher -Festlegen max zul. Verflüssigeraustrittstemp auf <=50°C (R502) ND: 2bar= -19°C, HD: 26bar= 60°C, Verflüssiger Pmax: 28bar= 64°C -Vor-&Nachlauf Ladepumpe normalerweise unnötig -Tempdiff über Verbraucher: 10K
